# time-series-analyzer

## WINDOWS:

To build locally, clone this repo then run:

```
set FLASK_APP=server.py
```
```
python -m flask run
```

Then open [`localhost:5000`](http://localhost:5000) in a browser.

## UNIX:

To build locally, clone this repo then run:

```
export FLASK_APP=server.py
```
```
python -m flask run
```

Then open [`localhost:5000`](http://localhost:5000) in a browser.
